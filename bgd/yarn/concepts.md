# Yarn

## Components 

- **Node Manger** 

- **Resource Manger** 
    * **Scheduler:** schedule job based on resources available within the cluster  
    * **Application Manager:** accepts job-submission and negotiate the first container with scheduler
      in order to create **Application Master**


- **Queue** 
   - *FIFO*
   - *Capacity Scheduler*
   - *Fair Scheduler*
