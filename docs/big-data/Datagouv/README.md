# Data Gouverance

## What's Data gouvernace ? 

Data function  to  ensure  the quality, integrity, security, and usability of the data collected by an organization.
Data must be secure meaning: 
- It's accessed only pemitted users and in permitted way
- It's auditable, meaning all access, including changes are logged 
- It's compliant with regulations

The purpose of data gouv is to enhance trust in data.
