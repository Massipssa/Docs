# Tools

## AWS CLI

- Install with pip
 ```pip install --user awscli```
- Configure
  - Add aws to path: ```export PATH=$PATH:$HOME/.local/bin```
  - Add secret key: ```aws configure```
- Credential will be added in the file ```$HOME/.aws/credentials```
